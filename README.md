# 🗣️ Pronunciation Mistake Detector

An Android app that detects pronunciation mistakes using speech-to-text technology. Built using Java, Android Studio, and Google’s Speech Recognition API.

## 🔍 Features

- 🎙 Records user's speech input
- 📄 Converts speech to text
- ✅ Compares with correct text input
- ❌ Highlights mispronounced words
- 📱 Simple and clean UI

## 🚀 How It Works

1. User enters a sentence
2. App records and converts speech to text
3. It compares spoken text vs typed text
4. Feedback is shown for any mismatches

## 🛠 Tech Stack

- Java
- Android Studio
- Google Speech-to-Text API
